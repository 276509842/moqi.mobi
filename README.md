##Moqi Mobile CMS

效果页面: [http://cms.moqi.mobi](http://cms.moqi.mobi)

####基本依赖库

 - RequireJS
 - Underscore
 - jQuery 


####模板及显示

 - Mustache 
 - Pure CSS 
 - Pagedown	
 
##如何运行

1. 在目录中运行一个HTTP服务器 
如

      python -m SimpleHTTPServer 8000  
或者
 
      mongoose
等等。

2. 修改``configure.conf``
